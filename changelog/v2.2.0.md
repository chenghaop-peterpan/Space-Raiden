# v2.2.0 — Dash 速度爆發 + 軌跡特效

**日期：** 2026-04-24
**類型：** minor

---

## 主題

Dash 機制從「瞬間傳送」改為「衝量速度爆發」：啟動後給予初速 80px/frame（player.speed 的 16 倍），每幀衰減 × 0.6，約 6 幀內自然停下，總位移 ~200px。同步加入速度流光特效。

---

## 修改

### Dash 機制（`fireDash()`）

| | 舊版（瞬移） | 新版（衝量爆發） |
|--|--|--|
| 實現方式 | 直接修改 player.x/y | 設定 dashImpulseX/Y，每幀更新 |
| 移動幀數 | 1 幀（瞬間） | ~6 幀（衰減停止） |
| 移動距離 | 固定 56px | ~200px（dashSpeed=80） |
| 視覺感 | 傳送感 | 衝刺感 |

### 衝量衰減公式

```
初速：dashImpulseY = -dashSpeed（向上）
每幀：dashImpulseY *= 0.60
停止：|impulse| < 0.8 → dashActive = false
```

### 速度流光特效

Dash 啟動時生成 10 條流光（`dashStreaks[]`）：
- 隨機分布於玩家底部附近
- 向下延伸（與衝刺方向相反），模擬加速噴射感
- 每幀 life -= 0.10，約 10 幀淡出
- 渲染：青色（`#0ff`）帶 shadowBlur

### 設定面板新增「衝刺速度」滑桿

- 範圍：30–120，步進 10，預設 80
- 即時生效，無需重啟遊戲

### 新增全域變數

```js
dashConfig.dashSpeed  = 80    // 初始衝量
let dashActive        = false // 衝量是否進行中
let dashImpulseX/Y    = 0     // 當前衝量值
let dashStreaks        = []    // 流光粒子陣列
```
