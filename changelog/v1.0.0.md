# [v1.0.0] - 2026-04-04

## 主題

初始版本：太空梭躲避隕石遊戲本體 + Playwright 測試套件建立

---

## 新增

### 遊戲
- `space_dodge.html`：HTML5 Canvas 太空射擊遊戲（單檔，無外部依賴）
  - 玩家移動、雷射發射、隕石生成與碰撞
  - 分數系統、等級系統、生命值、無敵時間
  - 爆炸粒子特效、畫面震動、星空背景

### 文件
- `README.md`：專案介紹、操控方式、遊戲機制、測試規範、檔案結構
- `docs/spec.md`：技術規格（狀態機、物理參數、渲染管線）
- `docs/testSpec.md`：測試規範（Function Test / Integration Test 分類定義）

### 測試
- `tests/conftest.py`：本機靜態伺服器 fixture（port 8765）
- `tests/test_ui.py`：UI Tests — 遊戲狀態、隕石、雷射、爆炸特效（10 項）
- `tests/test_api.py`：API Tests — DOM 狀態、鍵盤操作（10 項）

### 環境
- `.venv/`：Python 3.14.3 虛擬環境
- `.gitignore`：排除 .venv、\_\_pycache\_\_、report.html
