# v1.9.0 — localStorage 持久化

**日期：** 2026-04-23
**類型：** minor

---

## 主題

所有 LOG（Benchmark 歷史、Epoch 摘要、Input Log）自動儲存至 `localStorage`，頁面重新整理後資料不丟失。

---

## 新增

### 儲存機制
- `sr_runs`：`runHistory[]` 全量 JSON，每局結束後自動更新
- `sr_epochs`：`epochHistory[]` 全量 JSON，每個 Epoch 完成後自動更新
- `sr_inputlog`：最後一局的 `inputLog[]` 快照，每局結束後覆寫

### 函數
- `loadStorage()`：頁面啟動時從 localStorage 還原三組資料
- `_saveRuns()` / `_saveEpochs()` / `_saveInputLog()`：各自的 try/catch 儲存 helper
- `updateStorageInfo()`：計算並顯示「N 場 / X KB」於設定面板
- `clearStorage()`：移除三個 key，重置 runHistory / epochHistory / inputLog，更新面板

### HTML
- 設定面板 Log tab 新增「本地儲存：N 場 / X KB」顯示列
- 設定面板 Log tab 新增「✕ 清除本地儲存」紅色按鈕（含 confirm 確認）

---

## 觸發點整理

| 時機 | 動作 |
|------|------|
| 頁面載入 | `loadStorage()` → 還原三組資料 |
| `recordRun()` | `_saveRuns()` + `_saveInputLog()` + `updateStorageInfo()` |
| `finalizeEpoch()` | `_saveEpochs()` + `updateStorageInfo()` |
| 清除按鈕 | `clearStorage()` |
