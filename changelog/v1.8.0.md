# v1.8.0 — 統一設定面板 + Input Log + AI Terminal

**日期：** 2026-04-19
**類型：** minor

---

## 主題

以單一 `H` 鍵設定面板取代分散的 H/R/B/E 快捷鍵，並新增 Input Log（逐幀按鍵記錄）與 AI Terminal（即時決策捲動顯示）。

---

## 新增

### 設定面板（`#ctrl-panel`）
- `H` 鍵開啟面板，同時自動暫停遊戲；再按 `H` 或 `ESC` 關閉並恢復
- 三個 tab：**AI 參數** / **視覺化** / **Log**
- AI 參數 tab：遊戲中即時調整 aiParams（安全距離/瞄準容差/攻擊性/預測幀數/反應延遲）並同步原有 ai-panel 顯示值；支援策略切換
- 視覺化 tab：Debug HUD / 安全距離圓圈 / AI Terminal 的開關
- Log tab：Input Log 開關、Benchmark 面板開關、兩個 CSV 匯出按鈕

### Input Log
- `update()` 每幀記錄：幀號、按鍵、玩家位置 (X/Y)、位移 (dX/dY)、距離、事件
- 事件標記：`fireDash()` → `'DASH'`、`fireLaser()` → `'LASER'`、碰撞 → `'HIT'`
- 上限 7200 筆（約 2 分鐘），`startGame()` 時重置
- `#input-log-panel`：顯示最近 200 筆、摘要統計（DASH/LASER/HIT 次數）
- `exportInputLogCSV()`：匯出完整記錄為 CSV

### AI Terminal（`#ai-terminal`）
- 位於 canvas 右側（`#canvas-row` flex 佈局）
- 每幀 append 一行：幀號 / 決策狀態 / 指令 / 最近隕石距離
- 狀態顏色編碼：Idle 藍 / Tracking 黃 / Firing 綠 / Evading 橘 / Dodging 紅
- 自動捲到最新行，保留最近 300 筆
- `startGame()` 時重置

## 移除

| 舊快捷鍵 | 舊功能 | 新入口 |
|---------|--------|--------|
| `H` | Toggle Debug HUD | 設定面板 → 視覺化 tab |
| `R` | Toggle Safety Radius | 設定面板 → 視覺化 tab |
| `B` | Toggle Benchmark 面板 | 設定面板 → Log tab |
| `E` | 匯出 Benchmark CSV | 設定面板 → Log tab 按鈕 |

hint 列更新為：`方向鍵 / WASD 移動　　Space 發射雷射　　Shift：Dash　　H：設定`
