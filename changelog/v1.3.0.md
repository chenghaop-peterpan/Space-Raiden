# v1.3.0 — Benchmark 基礎設施

**日期**：2026-04-05

## 主題

為 AI 研究加入完整的績效評估系統（Benchmark），涵蓋視覺化趨勢圖、互動歷史表格與 CSV 匯出三大功能。

---

## 新增

### Plan A — Canvas 趨勢圖
- `drawBenchmarkChart()`：gameover 畫面下方自動繪製最近 10 場分數長條圖
- 當場高亮為青色，歷史場次為綠色，各 bar 上方顯示分數

### Plan B — HTML 歷史表格（`#benchmark-panel`）
- `B` 鍵切換浮層面板（`benchmarkVisible` flag）
- 摘要列：場次數、平均分數、最高分數
- 表格欄位：`#`、分數、等級、命中率（%）、時長（s）、Idle / Track / Fire / Evade / Dodge（%）
- 最新一場以黃色標示；`✕ 關閉` 按鈕可收起

### Plan C — CSV 匯出
- `exportCSV()`：點擊「↓ 匯出 CSV」或按 `E` 鍵下載 `benchmark.csv`
- 欄位：Run, Score, Level, Accuracy(%), Duration(s), Idle(%), Tracking(%), Firing(%), Evading(%), Dodging(%)

### 資料收集
- `runHistory[]`：每局結束後由 `recordRun()` 自動追加
- `stateFrameCounts`：AI 五種決策狀態的幀數分布，每幀於 `aiUpdate()` 末尾累加
- `renderBenchmarkPanel()`：更新 HTML 表格與摘要統計

---

## 修改

- `startGame()`：重置 `stateFrameCounts`
- `update()` gameover setTimeout：呼叫 `recordRun()` 記錄本局
- `aiUpdate()`：新增 `stateFrameCounts[aiDecisionState]++`
- keydown handler：新增 `B`（切換 benchmark 面板）、`E`（匯出 CSV）按鍵
- `draw()`：gameover 區塊末尾呼叫 `drawBenchmarkChart()`
- `#hint` 文字更新，加入 H / B / E 快捷鍵提示
