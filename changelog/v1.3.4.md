# v1.3.4 — GitHub Actions CI 優化

**日期**：2026-04-06

## 主題

升級 CI workflow：加入 Playwright browser caching、pip caching、觸發條件精簡化。

---

## 修改

- `.github/workflows/pytest.yml`：
  - Python 版本改為 `3.11`（CI 穩定性）
  - `setup-python` 加入 `cache: 'pip'`（快取 pip 下載）
  - 新增 `python -m pip install --upgrade pip`
  - 新增 Playwright browser caching（`actions/cache@v4`，key 綁定 playwright 版本）
    - cache miss：`playwright install chromium --with-deps`
    - cache hit：`playwright install-deps chromium`（只裝 system libs）
  - 測試指令改為 `python -m pytest`（確保使用正確環境）
  - 觸發條件移除 `feat/**` push（feature branch 僅在 PR to main 時觸發）
