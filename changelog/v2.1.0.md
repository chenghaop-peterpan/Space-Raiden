# v2.1.0 — 策略對比模式（Compare Mode）

**日期：** 2026-04-24
**類型：** minor

---

## 主題

新增「策略對比」功能：選定策略 A + 策略 B + 局數 N，系統自動循序執行，最後並排顯示 5 項指標，勝者標綠，敗者標紅。

---

## 新增

### 策略對比面板（`#compare-panel`）

**操作流程：**
1. H 鍵開啟設定面板 → Log 標籤 → ⚔ 策略對比
2. 選擇策略 A / 策略 B、設定局數（1–20）
3. 點「▶ 開始對比」
4. 系統自動跑 N 局 A → N 局 B → 彈出結果

**對比指標：**

| 指標 | 說明 |
|------|------|
| 平均分 | N 局平均得分，越高越好 |
| 最高分 | N 局最高單場分數 |
| 標準差 | 穩定性，越低越好（唯一反向指標） |
| 均存活 | 平均存活秒數，越高越好 |
| 命中率 | 平均射擊命中率，越高越好 |

勝者欄位標綠（`#4f4`），敗者標紅（`#f66`），平局無色。

### 進度狀態列

對比執行期間，面板內即時顯示：
```
策略 A（威脅迴避）進行中… 3/5
策略 B（軌跡預測）進行中… 1/5
對比完成（各 5 局）
```
