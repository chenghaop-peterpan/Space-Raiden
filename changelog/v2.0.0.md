# v2.0.0 — 軌跡預測型 AI 策略

**日期：** 2026-04-23
**類型：** major

---

## 主題

新增第四種 AI 策略「軌跡預測」，採用本質不同的決策邏輯：不等隕石靠近才反應，而是隕石一出現就計算其落點 X，提前側移閃開。

---

## 新增

### AI 策略：`trajectory`（軌跡預測）

**決策邏輯：**
```
對每顆隕石：
  frames = (player.y - asteroid.y) / asteroid.vy   // 落到玩家 Y 需要幾幀
  landingX = asteroid.x + asteroid.vx * frames      // 那時候隕石在哪個 X
  if |landingX - player.x| < (r + 36):              // 落點會打到我？
    urgency = dangerW / frames                       // 越近越緊急
    → 找最緊急威脅，往反方向移
```

**vs 現有反應式策略：**
| | 反應式（既有） | 軌跡預測（新） |
|--|--|--|
| 觸發條件 | 距離 < safetyRadius | 落點橫向偏差 < r+36 |
| 移動時機 | 隕石已接近才動 | 隕石剛出現就判斷 |
| 移動方式 | 緊急閃躲 | 小幅提前側移 |

**無威脅時：** 往畫面下半部中央靠攏（同防守型）
**射擊：** 與其他策略相同，隕石在上方且橫向對齊時射擊
**Dash：** 落點距離玩家 < 25 幀時自動觸發
