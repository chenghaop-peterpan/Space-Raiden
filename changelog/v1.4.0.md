# v1.4.0 — Skills + MCP 基礎建設

**日期：** 2026-04-06

## 主題

建立 Claude Code 開發工具基底：三個 slash commands 自動化重複流程，GitHub MCP 直接操作 GitHub。

## 新增

- **`.claude/commands/new-version.md`** — `/new-version` 指令：從 main 建立新 branch
- **`.claude/commands/test.md`** — `/test` 指令：一鍵跑全套測試
- **`.claude/commands/release.md`** — `/release` 指令：全自動 release 流程（含確認步驟）
- **`.mcp.json`** — GitHub MCP 設定（`@modelcontextprotocol/server-github`，需 `GITHUB_TOKEN` 環境變數）

## 修改

- **`.gitignore`** — 從忽略整個 `.claude/` 改為只忽略 `settings.local.json`，讓 commands 可提交
- **`CLAUDE.md`** — 新增「可用 Slash Commands」section，列出三個指令的用法
- **`docs/roadmap.md`** — 補齊 v1.3.6、v1.3.7、v1.4.0 完成記錄

## 使用方式

| 指令 | 用法 |
|------|------|
| `/new-version` | `/new-version feat/v1.5.0` |
| `/test` | `/test` |
| `/release` | `/release`（執行前會顯示確認摘要） |

**GitHub MCP 一次性設定：**
1. 在 GitHub 建立 Personal Access Token（scope: `repo`, `workflow`）
2. 設定系統環境變數：`GITHUB_TOKEN=<token>`
