## 新增

### 測試
- `tests/test_unit.py`：Unit Tests（白箱，8 項）— 純公式與算法驗算
  - level 公式邊界值（score 0/299/300/600/900）
  - 玩家碰撞閾值（a.r + 16）、雷射碰撞閾值（a.r + 4）
  - 邊界 clamp 公式（x、y 軸）
  - 分數計算（小隕石 +15、大隕石 +30）
  - 生成間隔公式 max(10, 28 - level×3)

- `tests/test_regression.py`：Regression Tests（黑箱，6 項）— 邊界防守
  - 無敵期間同幀兩顆隕石只扣一次血
  - dead state 按 Space 不發射雷射
  - gameover state 分數不自動增加
  - dead → 1200ms → gameover 狀態轉換
  - gameover 按 Space 完整重置遊戲
  - invincible 每幀遞減 1

### 設定
- `pyproject.toml`：pytest marker 統一定義（取代 pytest.ini）
  - smoke / unit / functional / integration / regression
