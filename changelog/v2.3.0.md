# v2.3.0 — 即時雙 AI 對比介面（Live Dual Compare）

**日期：** 2026-04-25
**類型：** minor

---

## 主題

全螢幕「即時雙 AI 對比」介面：兩種策略同時各自跑遊戲，搭配折線圖直觀比較每個 Epoch 的分數差異。

---

## 新增

### 即時雙 AI 對比介面（`#dual-compare-panel`）

**進入方式：** H 鍵設定 → Log 標籤 → ⚡ 即時雙 AI 對比

**版面：**
```
[Terminal A] [Canvas A 65%] [Canvas B 65%] [Terminal B]
         折線圖（X=Epoch，Y=分數，A=青色，B=橘色）
```

**操作：**
1. 選策略 A / 策略 B（各自下拉選單）
2. 設定 Epoch 數（1–50，預設 10）
3. 點「▶ 開始」→ 雙方同時跑遊戲
4. A 和 B 都死亡後算一個 Epoch，自動重啟下一個
5. 折線圖即時更新，勝者以粗體標示

### 折線圖

- X 軸：Epoch 編號
- Y 軸：分數（自動縮放）
- A 線：青色（`#0ff`）；B 線：橘色（`#fa0`）
- 格線、座標標籤、圖例、均分統計、領先標示

### State Swap 架構

技術核心：`saveState(gs)` / `loadState(gs)` 每幀交替執行兩份快照，共用同一份 `update()` / `draw()` 邏輯，只需 `const ctx → let ctx` 即可切換渲染目標。

**Snapshot 涵蓋 26 個變數：** frameCount, state, gameMode, score, lives, level, player, asteroids, lasers, explosions, particles, dashGhosts, dashStreaks, laserCooldown, invincible, shake, dashCooldown, dashActive, dashImpulseX/Y, currentStrategy, aiDecisionState, stateFrameCounts, shotsFired, shotsHit, aiTermLog
